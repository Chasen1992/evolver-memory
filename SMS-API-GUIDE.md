# 📱 短信API获取指南

## 为什么需要短信API？

用于注册平台账号时接收验证码：
- 掘金账号注册
- V2EX账号注册
- 博客园账号注册
- 其他需要手机号验证的平台

---

## 方案1：阿里云短信服务（推荐）

### 网址
https://www.aliyun.com/product/sms

### 价格
- 验证码短信：约 ¥0.045/条
- 1000条约 ¥45元
- 新用户有免费额度

### 获取步骤

#### 第1步：注册阿里云账号
1. 访问 https://www.aliyun.com
2. 点击"免费注册"
3. 使用支付宝或淘宝账号快速登录
4. 完成实名认证（需要身份证）

#### 第2步：开通短信服务
1. 登录阿里云控制台
2. 搜索"短信服务"
3. 点击"立即开通"
4. 选择"按量付费"（不用不花钱）

#### 第3步：获取API凭证
1. 进入短信服务控制台
2. 点击"AccessKey管理"
3. 创建AccessKey
4. 记录：
   - AccessKey ID
   - AccessKey Secret

#### 第4步：申请短信签名和模板
1. 提交短信签名（如"进化者"）
2. 提交短信模板（如"您的验证码是${code}"）
3. 等待审核（通常1-2小时）

## 使用示例代码

```python
import requests

# 聚合数据短信API示例
def send_sms(phone, code):
    url = "http://v.juhe.cn/sms/send"
    params = {
        "mobile": phone,
        "tpl_id": "您的模板ID",
        "tpl_value": f"#code#={code}",
        "key": "您的AppKey"
    }
    response = requests.get(url, params=params)
    return response.json()

# 使用
result = send_sms("13800138000", "123456")
print(result)
```
